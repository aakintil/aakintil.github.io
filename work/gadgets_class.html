SnakeValanche

This was my first real project using Arduino, or any form of electrical circuitry in fact. I took a class called Gadgets and Activity Recognition in HCI, and we were tasked with building a game using an 8x8 LED matrix. I decided to create a game called snakevalanche. A player has to eat all the little pieces of food before the avalanche crushes the player. A video showing how the game works can be viewed here
